# Geofence Test

Run Api
```bash
cd geoapi
# With Pipenv
pipenv install
# With PIP (Virutal environment needs to be created)
pip install -r requirements.txt
# Run
python manage.py makemigrations
python manage.py migrate
python manage.py runserver
```
Run front:
```bash
cd geoapp
npm install
npm start
```
